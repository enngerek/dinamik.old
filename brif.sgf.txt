<br/>
OPERASYON DİNAMİK
<br/>
Görev kısaca Altis Haritası genelinde rastgele oluşturulan bir seri yan görevden oluşmaktadır.
<br/>

<br/>
Genel Özellikleri
<br/>

<br/>
- Görevlerdeki bot sayısı ve araç dengesi 20 kişilik bir oyuncu grubuna denk gelecek şekilde ayarlanmıştır.
<br/>
- Oluşturulacak görev kendi içinde araç, gereç sayısı ve türü ile görev yeri açısından dinamiktir.
<br/>
- Her defasında bir yan görev türü oluşturulmakta bu görev tamamlandıktan sonra yenisi havuzdan rastgele seçilmektedir.
<br/>
- 4 kişilik Düşman Oyuncu slotu vardır.İstenirse 4 oyuncu bu slotlarda oyuna girerek karşı tarafın görevleri tamamlamasını engellemeye çalışabilir.(COOP-TVT)
<br/>
- INS revive : Yaralı oyuncu markerı, diğer oyuncu veya üsten respawn imkanı
<br/>
- Virtual AmmoBox System (VAS)
<br/>
- Virtual Vehicle Spawner (VVS)
<br/>
- Rastgele Hava Durumu
<br/>
- Takım liderlerinin kullanabileceği logistik destek sistemi 
<br/>
- Kırmızı takım için HALO jump
<br/>

<br/>
Yan Görev Türleri :
<br/>

<br/>
- Kasabayı işgal eden direnişçileri etkisiz hale getir :
<br/>
Görevi tammalamak için kasaba veya şehirdeki tüm direnişler öldürülmelidir.
<br/>

<br/>
- Deniz mayınlarını etkisiz hale getir :
<br/>
Görevi tamamlamak için deniz mayınları etkisiz hale getirilmelidir.Oyun motorunun kısıtlaması olarak sadece Patlayıcı uzmanı slotu mayınları etkisiz hale getirebilir.
<br/>

<br/>
- Düşen uçağımızın pilotunu enkaz alanından kurtar.
<br/>
Görevi tamamlamak için pilot bulunup Karargaha getirilmeli ve enkaz alanının 500m çevresindeki tüm düşman askerleri öldürümelidir.Enkaz alanı haritada işaretlidir.Kayıp pilot enkaz alanının 200m çevresinde aranmalıdır.
<br/>

<br/>
- Düşürülen UAV enkazını bularak harddiskini kopyala:
<br/>
Görevi tamamlamak için haritada yeri işaretli olan enkaza ulaşılmalıdır.Enkaz yanında iken aktif olacak action menu'den 'Harddiki Kopyala' seçilmeli ve 500m çevredeki düşman askerleri öldürülmelidir.
<br/>

<br/>
- Düşman denizaltı aracına sabotaj :
<br/>
Görevi tamamlamak için denizaltının demirlediği limana ulaşılmalı denizaltının yanında iken aktif olacak action menüden 'Patlayıcıları Yerleştir' seçilerek Denizaltı patlatılmalı ve 700m civardaki düşman askerleri öldürülmelidir.
<br/>

<br/>
- Rehin alınan gazeteciyi kurtar :
<br/>
Görevi tamamlamak için gazeteci bulunarak Karargaha getirilmelidir.Rehine, haritada işaretlenmiş dairesel marker alanın içindeki binalarda aranmalıdır.
<br/>

<br/>
- Silah taşıyan konvoyu imha et :
<br/>
Görevi tamamlamak için konvoydaki tüm askerler öldürülmelidir.Konvoy 7 araçtan oluşmaktdır.Dikkat ! Bazen araçlar yolda veya spawn oldukları noktalarda objelerde takılabiliyor.7 araç saydığınızdan emin olun.Aksi halde başlangıç noktasına doğru araçları bulup imha etmeniz gerekebilir.
<br/>

<br/>
- Tıbbi Tahliye :
<br/>
Görevi tamamlamak için yaralı asker karargaha getirilmelidir.Yaralı askerin yanında iken aktif olacak action menüden 'araca yükle' seçilerek asker 10m alandaki en yakın araca yüklenecektir.Aracınızı yaralı askere yakın tutmaya dikkat edin.
